D-тригер (D-Latch)

Що таке D-тригер?

D-тригер (Data Latch) — це цифровий елемент пам’яті, який може зберігати один біт інформації. Він має вхід даних (D), сигнал дозволу (EN) та два виходи (Q і Q').

Принцип роботи

Коли EN = 1, вихід Q повторює значення D. Коли EN = 0, тригер утримує попередній стан.

Таблиця істинності

EN (Дозвіл) D (Дані) Q (Вихід) Q' (Інверсний вихід) Стан
1 0 0 1 Q повторює D
1 1 1 0 Q повторює D
0 X Попередній стан Інверсія попереднього стану Утримання

Ключові характеристики

  • Прозорий: При EN=1 вихід миттєво повторює вхідні дані
  • Пам’ять: При EN=0 утримує попереднє значення
  • Простота: Реалізується кількома логічними вентилями

Застосування

  • Реєстри для зберігання даних
  • Буфери вводу-виводу
  • Побудова складніших тригерів (D-Flip-Flop)
